putDouble

open override fun putDouble(key: String, value: Double)

Saves a preference value of type Double

Parameters

key

This is the uniqueId with which the preference value will be saved, this is also what should be used to retrieve the saved value from preference by passing it to getDouble method call

value

This is the exact Double value that you want to save into preferences.
