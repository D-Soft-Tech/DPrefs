putObject

open override fun <T> putObject(key: String, value: T)

Save an object as a preference


Sample Usage

{
"firstName": "Adebayo",
"lastName": "Oloyede",
"age": 31,
"address": "Oyo state, Nigeria",
"latitude": 3.3456987,
"longitude": 7.432765,
"isLoggedIn": true
}
Content copied to clipboard

The `UserInfo` data class can be modelled from the above `json` payload. like so:

// UserInfo Model
data class UserInfo(
val firstName: String,
val lastName: String,
val age: Int,
val address: String,
val latitude: Double,
val longitude: Double,
val isLoggedIn: Boolean
)
Content copied to clipboard

Sample Usage `Kotlin`

// Instantiate an object of UserInfo
val userA = UserInfo("Adebayo", "Oloyede", 31, "Oyo state, Nigeria", 3.3456987, 7.432765, true)

// save userA as preference
val objectKey = "OBJECT_VALUE_KEY"

DPrefs.putObject(objectKey, userA)

Sample Usage `Java`

// Instantiate an object of UserInfo
UserInfo userA = new UserInfo("Adebayo", "Oloyede", 31, "Oyo state, Nigeria", 3.3456987, 7.432765, true);

// save userA as preference
String objectKey = "OBJECT_VALUE_KEY";
DPrefs.INSTANCE.putObject(objectKey, userA);

Parameters

key

This is the uniqueId with which the object value will be saved, this is also what the saved object will be accessed by from preference by passing it to getObject method call

value

This is the object that you want to save into preference.

See also

DPrefs.initializeDPrefs
DPrefsException

Throws

DPrefsNotInitializedException

if the library has not been initialized.

DPrefsKeyAlreadyExistsException

if the key already exists in preferences.
