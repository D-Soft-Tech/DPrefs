getObject

open override fun <T> getObject(key: String, type: T): Any?

Retrieves an object from preferences.


Sample Usage `Kotlin`

val key = "OBJECT_VALUE_KEY"
val savedObject = DPrefs.getObject(key, UserInfo::class)
Content copied to clipboard

Sample Usage `Java`

String key = "OBJECT_VALUE_KEY";
UserInfo savedObject = DPrefs.INSTANCE.getObject(key, UserInfo.class);
Content copied to clipboard

Return

An object of type T which was saved or null if no value exists for the key

Parameters

key

The key with which you want to get the object from the preferences.
Note. This is also the key with which the object was saved.

type

This is the type of the object you want to retrieve from the preferences. Note. This is also the type you must have saved initially.
